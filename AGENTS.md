# SLB QGIS Plugin

## Environment

* Windows 11
* QGIS Desktop
* MCP Plugin aktif
* Git: Laragon Git
* Push menggunakan PowerShell
* Git Credential Manager sudah terpasang

## Rules

* Jangan gunakan Bash git untuk push
* Gunakan PowerShell untuk git push
* Jangan buka GUI saat headless test
* Jangan commit sebelum headless test PASS

## Workflow

1. Implement
2. Headless test
3. Commit
4. Push
5. Report hash
